# Lab 10: Forensic Time Stomping

**Module:** M05 | **Course:** CYBR C160 | **Date:** November 18, 2025

## Tools Used
- Windows PowerShell
- NDG NetLab (Windows 10 VM)

## What I Did
Used PowerShell to navigate to a forensic tools directory, manipulated
file timestamps using time stomping techniques, then analyzed the result
from both the attacker and defender perspective.

## Key Concepts
- Time stomping as an anti-forensics technique
- File timestamp manipulation via PowerShell
- Why timestamps cannot be trusted alone in investigations
- Cross-referencing multiple evidence sources

## Why This Matters
Attackers modify timestamps to blend malicious files in with legitimate
system files. Understanding this means knowing to corroborate timestamps
with multiple artifact sources during any forensic investigation.
