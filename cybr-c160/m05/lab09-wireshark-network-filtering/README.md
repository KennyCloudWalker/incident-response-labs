# Lab 09: Filtering Network Traffic with Wireshark

**Module:** M05 | **Course:** CYBR C160 | **Date:** November 18, 2025

## Tools Used
- Wireshark
- PCAP file (lab15.pcap)
- NDG NetLab (Windows 10 VM)

## What I Did
Opened a pre-captured network traffic file in Wireshark, used Protocol
Hierarchy to analyze traffic composition, applied display filters to
isolate specific protocols, and identified notable traffic patterns.

## Key Concepts
- PCAP file analysis
- Wireshark display filters
- Protocol Hierarchy statistics
- Network traffic pattern identification

## Why This Matters
Wireshark is the industry-standard packet analysis tool expected in
any networking or security role. Analyzing real capture files is
directly applicable to SOC analyst, network analyst, and IR positions.
