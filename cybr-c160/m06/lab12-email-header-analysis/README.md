# Lab 12: Email Header Analysis

**Module:** M06 | **Course:** CYBR C160 | **Date:** November 24, 2025

## Tools Used
- Manual email header analysis
- NDG NetLab (Windows 10 VM)
- Lab14 folder sample files

## What I Did
Loaded sample email files, extracted and analyzed raw email headers
to trace the delivery path, identify spoofed sender fields, examine
relay server IPs, and detect indicators of phishing.

## Key Concepts
- Email header structure and delivery path tracing
- Spoofed From fields vs actual originating server IP
- Relay server hop analysis
- Phishing indicator identification

## Why This Matters
Phishing is the number one initial access vector in real breaches.
Reading raw email headers to find where a message actually originated
is a daily SOC analyst task.
