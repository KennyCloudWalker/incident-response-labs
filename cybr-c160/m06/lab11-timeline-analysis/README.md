# Lab 11: Timeline Analysis with Plaso / Log2Timeline

**Module:** M06 | **Course:** CYBR C160 | **Date:** November 28, 2025

## Tools Used
- Plaso (log2timeline.exe, psort)
- Command Prompt (Administrator)
- NDG NetLab (Windows 10 VM)

## What I Did
Navigated to the Plaso directory via admin Command Prompt, ran
log2timeline.exe against a forensic disk image (lab9.001) to generate
a super-timeline output file, then used psort to filter and analyze
the timeline data.

## Key Concepts
- Super-timeline creation from disk images
- log2timeline and psort workflow
- Aggregating events from hundreds of artifact sources
- Chronological incident reconstruction

## Why This Matters
Timeline analysis is how investigators reconstruct exactly what happened
on a system and when. Building a super-timeline from a disk image is
advanced work rarely seen at the entry level.
