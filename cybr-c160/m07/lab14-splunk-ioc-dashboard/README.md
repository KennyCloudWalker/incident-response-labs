# Lab 14: Splunk Dashboard Creation for Indicators of Compromise

**Module:** M07 | **Course:** CYBR C160 | **Date:** December 2, 2025

## Tools Used
- Splunk Enterprise
- SPL
- Splunk Dashboard Builder
- NDG NetLab (Windows 10 VM)

## What I Did
Ran IoC-focused SPL search queries then built a custom Splunk
dashboard to display and monitor indicators of compromise in a
persistent visual format for ongoing team-wide detection.

## Key Concepts
- SIEM dashboard design for continuous monitoring
- IoC visualization in Splunk Enterprise
- Operationalizing detection for team-wide visibility
- Splunk panel and dashboard builder workflow

## Why This Matters
Building dashboards operationalizes detection so the whole team
monitors threats visually. Dashboard creation is specifically listed
in SOC analyst and security engineer job descriptions.
