# Lab 04: Capturing a Memory Image with FTK Imager

**Module:** M02 | **Course:** CYBR C160 | **Date:** October 29, 2025

## Tools Used
- AccessData FTK Imager 4.5.0
- NDG NetLab (Windows 10 VM)

## What I Did
Installed FTK Imager, used the Capture Memory function to create a
full RAM image of the virtual machine, configured destination path
and filename, and included the pagefile in the capture.

## Key Concepts
- Volatile memory forensics
- Memory image acquisition with FTK Imager
- Pagefile inclusion in memory captures
- Chain of custody for digital evidence

## Why This Matters
Malware often lives only in RAM and disappears on reboot. A memory
image captures running processes, injected code, and credentials
that would never appear in a disk image alone.
