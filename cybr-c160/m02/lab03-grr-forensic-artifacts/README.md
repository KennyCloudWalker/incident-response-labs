# Lab 03: Retrieving Forensic Artifacts with GRR Rapid Response

**Module:** M02 | **Course:** CYBR C160 | **Date:** October 29, 2025

## Tools Used
- GRR Rapid Response
- Microsoft Edge
- NDG NetLab (Windows 10 VMs)

## What I Did
Logged into the GRR web interface, searched for and selected target
host Workstation1, browsed the remote file system, and retrieved
forensic artifacts from the live machine without direct access.

## Key Concepts
- Remote forensic artifact collection
- GRR host enumeration and selection
- Live system investigation without direct login
- Enterprise-scale forensic tooling

## Why This Matters
GRR is used by Google and large enterprise security teams. Performing
remote forensics without touching the host directly preserves evidence
integrity and avoids alerting an attacker.
