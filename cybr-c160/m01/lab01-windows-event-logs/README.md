# Lab 01: Exploring Windows Event Logs with Event Viewer

**Module:** M01 | **Course:** CYBR C160 | **Date:** October 25, 2025

## Tools Used
- Windows Event Viewer
- NDG NetLab (Windows 10 VM)

## What I Did
Connected to Windows 10 Workstation 2 via NetLab, opened Event Viewer,
navigated the Windows Logs folders (Application, Security, Setup, System),
and identified user login events by their Event IDs.

## Key Concepts
- Windows Security log structure
- Event ID 4624 — successful user logon
- Navigating Event Viewer log categories
- Filtering events by ID and time

## Why This Matters
Event Viewer is one of the first tools a SOC analyst reaches for during
triage. Every investigation starts with understanding who logged in,
when, and from where.
