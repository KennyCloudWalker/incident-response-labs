# Lab 06: Kill Chain — Weaponization Stage

**Module:** M03 | **Course:** CYBR C160 | **Date:** November 7, 2025

## Tools Used
- Windows PowerShell (Administrator)
- NDG NetLab (Windows 10 VM)

## What I Did
Opened PowerShell with administrator privileges and executed commands
simulating the Weaponization stage of the Lockheed Martin Cyber Kill
Chain, documenting each step and its role in the attack lifecycle.

## Key Concepts
- Lockheed Martin Cyber Kill Chain framework
- Weaponization stage — payload preparation
- Elevated privilege execution via PowerShell
- Attacker methodology from the defender's perspective

## Why This Matters
Understanding each kill chain stage helps defenders know what to look
for and where to interrupt an attack. Speaking to the kill chain
framework is expected in most entry-level security roles.
