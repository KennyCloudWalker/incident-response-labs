# Lab 15: Threat Hunting with Splunk

**Module:** M08 | **Course:** CYBR C160 | **Date:** December 2, 2025

## Tools Used
- Splunk Enterprise
- SPL (EventID 4688 process execution queries)
- NDG NetLab (Windows 10 VM)

## What I Did
Ran process execution event searches using EventID 4688 filtered
for suspicious file types including .rar files, identified anomalous
process activity, and documented findings as part of a structured
threat hunt.

## Key Concepts
- Proactive threat hunting methodology
- EventID 4688 — Windows process creation logging
- Hunting for data exfiltration indicators
- Hypothesis-driven investigation in a SIEM

## Why This Matters
Threat hunting is proactive — going looking for compromise instead
of waiting for an alert. This skill separates deeper security
thinkers from alert-only analysts and is in high demand.
