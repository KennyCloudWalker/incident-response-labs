# CYBR C160 — Introduction to Incident Response

**Institution:** Coastline Community College
**Term:** Fall 2025
**Tools:** Splunk · GRR Rapid Response · FTK Imager · Wireshark · Plaso · PowerShell
**Lab Environment:** NDG NetLab (Windows 10 virtual machines)

---

## Course Overview

Hands-on introduction to digital forensics and incident response (DFIR) covering
the full IR lifecycle — evidence collection, artifact retrieval, log analysis,
threat hunting, and malware analysis.
