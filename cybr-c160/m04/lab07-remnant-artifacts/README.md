# Lab 07: Identifying Remnant Artifacts of a Compromised Computer

**Module:** M04 | **Course:** CYBR C160 | **Date:** November 11, 2025

## Tools Used
- Sysinternals Process Explorer (procexp64.exe)
- NDG NetLab (Windows 10 VM — pre-compromised)

## What I Did
Connected to a pre-compromised Windows 10 VM, launched Process
Explorer, identified a malicious process (mozillar.exe) disguised
as a legitimate browser process, and documented the indicators
of compromise.

## Key Concepts
- Process masquerading and name spoofing
- Malicious process identification with Process Explorer
- Remnant artifact analysis post-compromise
- Indicators of Compromise (IoCs)

## Why This Matters
Attackers name malicious processes to mimic legitimate software.
Recognizing process masquerading like mozillar.exe faking Mozilla
is a hands-on triage skill that stands out at the entry level.
